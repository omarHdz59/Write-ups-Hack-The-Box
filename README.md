# 🛠️ Herramientas Utilizadas Frecuentemente

Durante estas resoluciones, empleo un conjunto de herramientas estándar en la industria:

- Reconocimiento: `Nmap` , `WhatWeb`.
- Enumeración Web: `Gobuster`, `Burp Suite`.
- Explotación/Análisis: `Metasploit Framework` , `Searchsploit`.
- Ingeniería Inversa: `JD-GUI` (para archivos .jar).
- Escalada de Privilegios: `LinPEAS`, `Sudo -l`.
